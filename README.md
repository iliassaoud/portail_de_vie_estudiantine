# Exercice Portail Étudiant

Ceci est un simple exercice de développement web pour l'ENSA Marrakech. L'objectif était de pratiquer les techniques de mise en page de base et l'adaptabilité.

Le projet utilise :
- **CSS Grid & Flexbox** pour l'organisation de la structure.
- Un **design responsive** qui modifie la mise en page selon la résolution de l'écran.
